## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/yamneg96/NikeWebsite.git
   cd NikeWebsite
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```
